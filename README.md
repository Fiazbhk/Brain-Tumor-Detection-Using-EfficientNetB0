# Brain Tumor Detection Using EfficientNetB0

A deep learning system for classifying brain MRI scans into four categories using EfficientNetB0 with transfer learning. Deployed as an interactive web application using Streamlit.

**Author:** Muhammad Fiaz

---

## Overview

This project applies transfer learning to the problem of brain tumor classification from MRI images. The model is trained to distinguish between four classes: Glioma, Meningioma, No Tumor, and Pituitary. The trained model is served through a clean Streamlit interface that accepts image uploads and returns real-time predictions with per-class confidence scores.

---

## Classes

| Label | Class | Severity |
|---|---|---|
| 0 | Glioma | High |
| 1 | Meningioma | Moderate |
| 2 | No Tumor | None |
| 3 | Pituitary | Low to Moderate |

## Dataset

**Source:** Brain Tumor MRI Dataset by Masoud Nickparvar (Kaggle)

| Split | Images per Class | Total |
|---|---|---|
| Training | 1,400 | 5,600 |
| Testing | 400 | 1,600 |
| Total | 1,800 | 7,200 |

---

## Model Architecture

Base model: EfficientNetB0 pre-trained on ImageNet

```
EfficientNetB0 (frozen in Phase 1)
    GlobalAveragePooling2D
    BatchNormalization
    Dense(256, relu)
    Dropout(0.5)
    Dense(4, softmax)
```

### Training Strategy

**Phase 1:** Base model frozen, only classification head trained (lr = 1e-3)

**Phase 2:** Full network fine-tuned end-to-end (lr = 1e-5)

## Running Locally

**1. Create and activate a virtual environment**

```bash
python -m venv venv
venv\Scripts\activate        # Windows
source venv/bin/activate     # Mac / Linux
```

**2. Install dependencies**

```bash
pip install -r requirements.txt
```

**3. Run the app**

```bash
streamlit run app.py
```

The app will open at https://deepneuroscan.streamlit.app/
